# TCP 和 UDP 的区别

> https://zhuanlan.zhihu.com/p/108579426

|          | TCP                              | UDP                            |
| -------- | -------------------------------- | ------------------------------ |
| 可靠性   | 可靠                             | 不可靠                         |
| 连接性   | 连接（三次握手、四次挥手）       | 无连接                         |
| 报文     | 面向字节流                       | 面向报文                       |
| 效率     | 传输效率低                       | 传输效率低                     |
| 双工性   | 全双工                           | 一对一、一对多、多对一、多对多 |
| 流量控制 | 滑动窗口                         | 无                             |
| 阻塞控制 | 慢开始、阻塞避免、快重传、快恢复 | 五                             |
